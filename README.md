# CollabHub

CollabHub is a modern authentication interface for an Idea Pitching Platform, built using React, Vite, Tailwind CSS, and shadcn/ui.

## Features

- Responsive Login page
- Responsive Sign Up page
- Modern split-screen layout
- Clean UI built with shadcn/ui
- Tailwind CSS styling
- Reusable React components

## Tech Stack

- React
- Vite
- Tailwind CSS
- shadcn/ui
- JavaScript


## Project Structure

src/
─ assets/
─ components/
─ pages/
  ─ Login.jsx
  ─ Signup.jsx
─ App.jsx
─ main.jsx
